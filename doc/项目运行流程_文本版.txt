项目运行完整流程（文本版）
====================================================

【起点】用户输入命令
    │
    ├─ 仅说"更新价格" → 默认 mode=both（网价+图片/文档）
    ├─ 明确"只更新网价" → mode=web
    └─ 明确"只更新图片/文档" → mode=image_doc
    │
    ▼

【步骤1】前置准备
    ├── 从文件名提取：地点1（网价）、地点2（图片/文档）
    └── 创建运行产物目录：运行产物/<项目名>/
    │
    ▼

【步骤2】网价链路（mode=web 或 mode=both 时执行）
    │
    ├─ 2.1 登录网价站点
    ├─ 2.2 查询最新价格清单（按日期+时间排序）
    ├─ 2.3 生成清单Excel：[地点][日期]建筑钢材原料价格清单.xlsx
    ├─ 2.4 加载/复用厂家对照表（已有对照自动复用）
    │
    ├─ 2.5 【阻断点】存在新厂家待确认？
    │      ├─ 是 → 展示待确认明细 → 等待用户在对话中确认
    │      │       → 写入已确认对照表 → check_pending_mapping.py确认
    │      └─ 否 → 继续
    │
    ├─ 2.6 回写项目文件：G1(日期) / G3(盘螺) / G4(螺纹)
    └─ 2.7 页签处理：先清空后标红未匹配
    │
    ▼

【步骤3】图片/文档链路（mode=image_doc 或 mode=both 时执行）
    │
    ├─ 3.1 收集线下报价文件
    │      └─ 来源：线下报价/*.jpg *.png *.txt
    │
    ├─ 3.2 【本次修改】提取价格和库存
    │      ├─ .jpg/.png → MiniMax视觉识别（替代原OCR）
    │      │             → 输出：价格 + 库存状态（充足/告警/缺货）
    │      ├─ .txt      → 正则解析价格
    │      └─ 补充.txt  → 解析调整规则（下30/电议等）
    │
    ├─ 3.3 生成JSON结果文件
    │      └─ 运行产物/ocr价格提取_<厂家>.json
    │
    ├─ 3.4 加载/复用图片文档对照表（已有对照自动复用）
    │
    ├─ 3.5 【阻断点】存在新厂家待确认？
    │      ├─ 是 → 展示待确认明细 → 等待用户在对话中确认
    │      │       → 写入已确认对照表 → check_pending_mapping.py确认
    │      └─ 否 → 继续
    │
    ├─ 3.6 回写项目文件：H1(日期) / H3(盘螺) / H4(螺纹)
    │      └─ 注意：禁止修改页签颜色
    │
    └─ 3.7 【本次新增】库存颜色标注
           ├─ 3.7.1 清空"报价表"sheet原有颜色
           ├─ 3.7.2 建立映射：
           │         - 厂家名 → Excel列（读取第1行）
           │         - 规格+长度 → 行号（读取A-C列）
           ├─ 3.7.3 逐厂家标注单元格颜色：
           │         - 充足 = 蓝色
           │         - 告警（极少/少/XX件/XX配）= 黄色
           │         - 缺货（无货）= 红色
           │         - 只标第一列（价格列），不标网差列
           └─ 3.7.4 长江特殊处理：
                     - "厂内"开头的库存 → 标注T列
                     - "蚌埠"开头的库存 → 标注V列
    │
    ▼

【步骤4】生成报告
    │
    ├─ 4.1 生成总结报告 + 分阶段报告
    │
    └─ 4.2 【本次修改】输出结果概述
           ├─ 网价更新明细（G1/G3/G4）
           ├─ 图片/文档更新明细（H1/H3/H4）
           ├─ 【新增】库存颜色统计
           │      例：蓝色15个 / 黄色8个 / 红色2个
           ├─ 汇总统计表
           └─ 报告文件路径
    │
    ▼

【终点】返回结果给用户

修改3：结果概述增加库存统计

====================================================
沿用原有不变的关键规则
====================================================

✓ 厂家对照确认：新厂家必须经用户确认，禁止私自确认
✓ 页签颜色：网价"先清空后标红"；图片/文档禁止改页签颜色
✓ 库存颜色：仅标注"报价表"sheet，只标第一列，不标网差列
✓ 价格对比格式：必须使用"老值 → 新值"格式
✓ 待确认阻断：存在待确认时阻断，用户确认后继续
✓ 增量复用：已有对照表自动复用，只确认新厂家
