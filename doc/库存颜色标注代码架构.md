# 库存颜色标注代码架构

## 1. 整体架构

```
报价更新主流程 (pipeline.py)
    │
    ├─ 价格回写完成
    │
    ▼
库存颜色标注模块 (inventory_color.py)
    │
    ├─ 1. 清空原有颜色
    ├─ 2. 建立映射
    │   ├─ 厂家名 → 列号
    │   └─ 规格+长度 → 行号
    ├─ 3. 逐厂家标注
    │   ├─ 充足 → 蓝色
    │   ├─ 告警 → 黄色
    │   └─ 缺货 → 红色
    └─ 4. 保存文件
    │
    ▼
库存解析模块 (inventory.py)
    │
    ├─ 从JSON提取库存文本
    ├─ 解析库存状态
    └─ 返回库存项列表
```

### 2.1 inventory_color.py - 库存颜色标注

#### 主要函数

```python
def annotate_inventory(
    project_file: str,
    ocr_json_dir: str,
    sheet_name: str = "报价表"
) -> dict:
    """
    主入口函数：标注库存颜色
    
    流程：
    1. 打开项目Excel
    2. 清空原有颜色
    3. 建立映射（厂家→列，规格→行）
    4. 逐厂家标注颜色
    5. 保存文件
    
    返回：统计信息（蓝/黄/红数量）
    """

def _build_factory_map(sheet) -> dict[str, FactoryColumnMap]:
    """
    建立厂家名到列号的映射
    
    读取第1行，解析厂家名和对应列：
    - 每个厂家占2列（第一列=价格，第二列=网差）
    - 只标注第一列（价格列）
    
    特殊处理：
    - 长江：占用4列（T=厂内价格，U=厂内网差，V=蚌埠价格，W=蚌埠网差）
    """

def _build_inventory_row_map(sheet) -> dict[tuple, int]:
    """
    建立规格到行号的映射
    
    读取A-C列：
    - A列：钢材类型（一级钢/抗震三级钢）
    - B列：规格号（6, 8, 10, 12, 14, 16, 18, 20, 22, 25, 28, 32）
    - C列：长度（9/12米，盘螺为空）
    
    返回：{(类型, 规格, 长度): 行号}
    """

def _apply_inventory_colors(
    sheet,
    factory_map: dict,
    inventory_rows: dict,
    inventory_items: list[InventoryItem]
):
    """
    应用库存颜色
    
    颜色定义：
    - 充足：FF0070C0（蓝色）
    - 告警：FFFFC000（黄色）
    - 缺货：FFFF0000（红色）
    """
```

#### 数据结构

```python
@dataclass
class FactoryColumnMap:
    """厂家列映射"""
    name: str           # 厂家名（如"徐钢"）
    price_col: str      # 价格列（如"P"）
    netdiff_col: str    # 网差列（如"Q"）
    extra_cols: dict    # 额外列（如{"厂内": "T", "蚌埠": "V"}）

@dataclass
class InventoryItem:
    """库存项"""
    product: str        # 产品类型（螺纹/盘螺/线材）
    spec: str           # 规格（如"12E"）
    length: str         # 长度（9/12）
    status: str         # 状态（充足/告警/缺货）
    raw_desc: str       # 原始描述
```

### 4.1 长江双列映射

```python
# 长江占用4列
长江 = FactoryColumnMap(
    name="长江",
    price_col="T",      # 厂内价格
    netdiff_col="U",    # 厂内网差
    extra_cols={
        "厂内": "T",     # 厂内库存
        "蚌埠": "V"      # 蚌埠厂库库存
    }
)

# 标注逻辑
if inventory_spec.startswith("厂内"):
    col = "T"  # 厂内列
elif inventory_spec.startswith("蚌埠"):
    col = "V"  # 蚌埠列
else:
    col = "T"  # 默认厂内列
```

### 4.2 规格归一化

```python
def normalize_spec(inventory_spec: str) -> tuple[str, str, str]:
    """
    归一化库存规格
    
    输入："9米螺纹14E"、"厂内9m螺纹12E"、"盘螺8E"
    输出：(类型, 规格, 长度)
    
    规则：
    1. 提取数字 → 规格号

    """
```

### 4.3 模糊匹配

```python
# 厂家名模糊匹配
```

## 7. 注意事项

1. **只标第一列**：库存颜色只标注价格列，不标注网差列
2. **清空再标注**：每次标注前先清空原有颜色
3. **长江特殊**：厂内和蚌埠分别标注不同列
4. **模糊匹配**：厂家名和规格都支持模糊匹配
5. **优先级**：缺货 > 告警 > 充足（合并时）
6. **忽略未匹配**：报价表无此规格时，忽略不标注
